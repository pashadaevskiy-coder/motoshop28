# 🏍️ Эндуро мотоциклы - MOTO-SHOP

Современный продающий веб-сайт для продажи эндуро мотоциклов с интеграцией Telegram.

## 🚀 Технологии

- **React 18** - UI библиотека
- **Vite** - быстрая сборка и dev-сервер
- **Tailwind CSS** - утилитарный CSS фреймворк
- **Google Fonts** - Inter, Bebas Neue

## 📦 Установка и запуск

### 1. Установка зависимостей

```bash
npm install
```

### 2. Запуск dev-сервера

```bash
npm run dev
```

Открой http://localhost:3000 в браузере.

### 3. Сборка для production

```bash
npm run build
```

Собранные файлы появятся в папке `dist/`.

## 📱 Функциональность

### Основные секции

1. **Hero** - привлекающий внимание блок с фоновым изображением
2. **Каталог** - сетка карточек мотоциклов с фильтрами
3. **Почему мы** - 4 преимущества
4. **Отзывы** - отзывы клиентов
5. **Как купить** - 3 шага
6. **Финальный CTA** - призыв к действию
7. **Футер** - контакты и соцсети

### Фильтры каталога

- По цене (слайдер)
- По кубатуре (чекбоксы)
- По бренду (чекбоксы)

### Модалка товара

- Галерея изображений
- Полные характеристики
- Описание
- Кнопка "Заказать в Telegram"

### Интеграция с Telegram

Все кнопки "Купить" и "Заказать" ведут на Telegram-чат.

## ⚙️ Конфигурация

### Tailwind CSS

Кастомные цвета и настройки в `tailwind.config.js`:

```javascript
colors: {
  'bg-dark': '#000000',
  'bg-card': '#111111',
  'accent-orange': '#ff4500',
  // ...
}
```

### Telegram ссылка

Измените ссылку в файлах:

1. `src/data/motorcycles.js` - `TELEGRAM_LINK`
2. `src/components/Header/Header.jsx` - ссылки в навигации
3. `src/components/Hero/Hero.jsx` - CTA кнопки
4. `src/components/Catalog/MotorcycleCard/MotorcycleCard.jsx` - кнопка "Купить"
5. `src/components/ProductModal/ProductModal.jsx` - кнопка "Заказать"
6. `src/components/FinalCTA/FinalCTA.jsx` - финальная CTA

Замените `username` на ваш реальный Telegram username.

## 🚀 Деплой

### Netlify / Vercel

1. Соберите проект: `npm run build`
2. Загрузите папку `dist/` на хостинг
3. Настройте домен (опционально)

### GitHub Pages

1. Соберите проект
2. Перенесите файлы из `dist/` в ветку `gh-pages`
3. Настройте GitHub Pages
